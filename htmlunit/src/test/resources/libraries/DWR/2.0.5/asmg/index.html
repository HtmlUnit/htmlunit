Anti-spam mailto: Link Creator.

mailto: links allow you to give people a simple way to send you e-mail from a web page. Unfortunately they also allows spammers to harvest addresses. The solution is to use a simple script to generate mailto: links.

The following paragraph contains a mailto link generated by Javascript. It acts in the normal way, but if you check the source you will see that the address is obfusticated enough to be unreadable by spam robots:

Contact the helpdesk using:

Create Your Own

EMail Address:
Name:
 

Generated Links
